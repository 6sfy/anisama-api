anisama

CLI Anime VOSTFR/VF. Search, stream and download your anime from 6 French sources — from your terminal, or via the central API.

🌐

6 French Sources

Anime-Sama, MyFluneo, Voiranime, AnimesUltra, French-Anime & AnimoFlix.

🎯

Smart Search

Realistic fuzzy matching (rapidfuzz), title dedup, 6 cascading sources.

⚡

Centralized API

Server-side scraping + resolution. The client only consumes clean JSON.

▶️

Stream & Download

mpv, VLC, IINA, MoonPlayer, ImPlay or direct download.

📋

History & Continue

Resume watching at the exact episode, cross-source.

ℹ️

AniList Info

Scores, studios, synopsis, tags directly in the TUI.

Getting Started

Quick Start

Install and get running in 30 seconds.

bash
# Install with pip $ pip install anisama # Launch interactive TUI $ anisama # Or search directly $ anisama "one piece" -e 1-5

The interactive TUI launches automatically. ↑↓ to navigate, Enter to confirm, Esc to go back.

Getting Started

Installation

PyPI or source — your choice.

From PyPI

bash
$ pip install anisama

From Source

bash
$ git clone https://github.com/6sfy/anisama.git $ cd anisama $ pip install -e . $ anisama

Windows: use python -m src from the project root, or install with pip install -e .

API Server (optional)

If you want to host your own central API:

bash
$ cd api $ python run.py # Server starts on http://0.0.0.0:20100

API Reference

All JSON endpoints from the central API.

Base URL

url
https://anisama.sexcaller.lol

Endpoints

MethodEndpointDescription
GET/api/v2/search?q=QUERYSearch across all 6 sources. Returns deduped results with real fuzzy scores (0-100).
GET/api/v2/catalog?source=all&page=1&limit=50&q=TEXTBrowse the full catalog with filters, pagination and text search.
GET/api/v2/statsGlobal stats: counts by source, top genres, year range.
GET/api/v2/episodes?source=SRC&slug=SLUGGet episode list for an anime. Supports: anime-sama, myfluneo, voiranime, animesultra, frenchanime, animoflix.
GET/api/v2/resolve?url=URL&source=SRCResolve any embed URL to a direct video URL.
GET/api/v2/resolve-episode?source=SRC&slug=SLUG&num=NUMResolve a specific episode number. Uses absolute episode numbering for multi-season anime.
GET/api/v2/sourcesList all available sources with entry counts.
GET/api/v2/info?title=TITLEAniList info for an anime (score, studios, synopsis).
GET/player?url=URL&title=TITLEHTML player page with iframe.

Example: Search

bash
$ curl "https://anisama.sexcaller.lol/api/v2/search?q=naruto" # Returns deduped results with realistic scores

Example: Catalog

bash
$ curl "https://anisama.sexcaller.lol/api/v2/catalog?source=myfluneo&limit=100" $ curl "https://anisama.sexcaller.lol/api/v2/catalog?q=attack+on+titan"

Sources

Anime-Sama

Main source VOSTFR/VF.

Anime-Sama is the main source with ~1500 entries. The site uses external embed players:

  • vidmoly — video embed player
  • sendvid — direct MP4 player
  • sibnet — russian video host
  • smoothpre — embed player
  • streamwish — video host
  • oneupload — video upload

The multiple mirror system enables automatic fallback. Speed-testing picks the fastest mirror. Scan / Manhwa / Webtoon seasons are automatically filtered.

Sources

Voiranime

Secondary source VOSTFR/VF.

Voiranime provides an alternative catalog with its own internal video resolution system. ~960 entries merged into the main catalog via alt_source.

Sources

MyFluneo NEW

~5400 French entries.

MyFluneo (myfluneo.eu) is a new French source with ~5400 entries VOSTFR/VF. The site is built with Next.js:

  • vidmoly.biz — MP4 embed player
  • streamtape.com — embed player

Server-side scraped with Playwright (Next.js RSC). Episodes and resolutions are pre-cacheable.

Sources

AnimesUltra

~2600 entries via XML sitemap.

AnimesUltra (ww.animesultra.org) uses a sitemap.xml to expose its full catalog (~2600 URLs).

  • vidmoly, sendvid, sibnet, vidstream.pro

Resolution uses the data-embed attribute on server buttons. No dynamic JS required for the catalog.

Sources

French-Anime

Homepage scraping.

French-Anime (french-anime.com) lists recent animes on its homepage.

  • luluvdo.com — main host
  • vidmoly.biz, vidara.to

Resolution goes through iframe extraction from the episode page.

Sources

AnimoFlix

JS-heavy site with Playwright fallback.

AnimoFlix (animoflix.to) is a modern site with JavaScript rendering:

  • sendvid, sibnet.ru

The catalog is statically scraped via /catalogue/. Episodes use a Playwright fallback if static HTML is not enough. Server tabs are simulated via automated clicks.

Support

Troubleshooting

No results

Search uses fuzzy matching (rapidfuzz). Try shorter terms. The search cascades through 6 sources.

Player won't launch

Install mpv or VLC. Use anisama -p vlc "title" to specify the player.

Windows import error

Run python -m src from the root, or pip install -e . then anisama.

Blocked domain

Run anisama -U to force a refresh. Auto-detection of active domains.

API returns "Could not resolve"

Some sources use Playwright (MyFluneo, AnimoFlix). Make sure Chromium is installed on the API server.
